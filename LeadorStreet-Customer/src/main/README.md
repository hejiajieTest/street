                -css                    引用的css文件，其中index.css由scss编译得来，是所有自写样式的集合。
                -font                   字体
                -img                    引用的图片
                -js                     引用的js文件
                -sass                   各部分样式分别写入不同scss文件中，通过gulp编译成总的html.css文件
            --WEB-INF                 登录的配置文件
